Details

All you need to know about The Code Fellas Hackathon

“ Rules ”

Here are the Rules for the event

  1. Registration Fees Rs 50/- per team.
  2. Team Size : 2-3 Members (max : 3).
  3. Cash Prizes & Achievement Certificates for Top 3 Winners .
  4. Appreciation Certificates for Top positions in Hackathon other than winners.
  5. Prizes for Best Women Team.
  6. Participation Certificates for All .
  7. Project Submissions will be done online.
  8. Attendance will be given to those who will Register and Submit their Projects on time.

“ Themes ”

Here are the themes for the event

  1. Web Development : Static and dynamic website to solve problems creatively.
  2. App Development : Developing useful applications for mobile devices.
  3. Machine Learning : Solving real world problems by using optimized and effective ML algorithms.
  4. Internet of Things : Establishing a network of physical objects for connecting and exchanging the data with other devices and systems over the internet.
  5. Blockchain Technology : Creating a system of recording information in a way that makes it difficult to change,hack or cheat the system.
  6. Open Innovation : Any project based on innovative ideas or technologies to solve real time problems like Data Structures and Algorithm based projects, python projects e.t.c.
